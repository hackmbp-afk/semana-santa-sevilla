# Semana Santa Sevilla

Aplicación web desarrollada con Flask y MariaDB para gestionar hermandades de Semana Santa.

## Funcionalidades

- Crear hermandades
- Editar hermandades
- Eliminar hermandades
- Visualizar listado

## Tecnologías usadas

- Python (Flask)
- MariaDB
- HTML, CSS

## Instalación

1. Clonar repositorio:
   git clone https://github.com/TU_USUARIO/semana-santa-sevilla

2. Instalar dependencias:
   pip install -r requirements.txt

3. Ejecutar la aplicación:
   python cofradias.py

## Base de datos

El archivo `.sql` contiene la estructura y datos iniciales.

## Acceso

La aplicación está alojada en red local y es accesible desde Internet mediante redirección de puertos (port forwarding) configurada en el router.

Se ha utilizado un dominio dinámico (DuckDNS) para facilitar el acceso externo.
